Visibility of System Status


"The system should always keep users informed about what is going on, through appropriate feedback within reasonable time."

Jakob Nielsen

This means that when a user is interacting with a site, the site should provide visual cues to allow the user to know whether or not they are on the right track. This can include progress bars, active links, and bread-crumbs.


For this particular site, it never highlights where you are on the page, meaning it doesnâ€™t provide any visual cues such as active links or bread-crumbs. Although there is a consistent navigation bar at the top, there is no general directory to highlight which sub-section you are on. The site does have a banner indicating which part of the site you are on but the banner is inconsistent at times by not providing the accurate location.


The site also contains links that takes you to different pages. For example, clicking the library link takes you to another page but on this page, the layout is different and there is no longer a navigation bar or anything similar to relate back to the home site. This can confuse users to thinking if they are still on the Louisiana.edu site.
