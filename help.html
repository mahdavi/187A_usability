Help and documentation


"Even though it is better if the system can be used without documentation, it may be necessary to provide help and documentation. Any such information should be easy to search, focused on the user's task, list concrete steps to be carried out, and not be too large."

Jakob Nielsen

Although a good site does not need help and documentation, it is provided by sites to help the users in the event that they do. The Louisiana.edu website does not provide any form of help or documentation link anywhere on the site. Users will have to manually go to the search bar to search for either help or FAQ.


When users search for help or faq, the site does provide the user with adequate links. The help search provides users with many links, one of which is the IT help desk. This is helpful but could be better if this was itself a page on the Louisiana.edu website. The FAQ also answers the necessary questions but could also be better it it was its own page.
